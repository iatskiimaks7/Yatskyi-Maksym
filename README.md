# Бот для обліку особистих витрат

## Мета проєкту

Метою даного проєкту є створення консольного бота на мові **Python**, який дозволяє користувачу вести облік особистих витрат. Бот допомагає контролювати бюджет, додавати витрати, переглядати їх, аналізувати витрати за категоріями та перевіряти залишок коштів.

---

## Опис логіки та алгоритму роботи бота

Програма працює у консольному режимі та взаємодіє з користувачем через текстові команди.

Основні етапи роботи програми:

1. **Запуск програми**

   * При запуску бот вітає користувача.
   * Дані про бюджет і витрати зчитуються з файлу `finance_data.json`.

2. **Введення команд**

   * Користувач вводить команду у консолі.
   * Програма перевіряє команду та виконує відповідну функцію.

3. **Обробка команд**

   * встановлення бюджету;
   * додавання витрат;
   * перегляд витрат;
   * фільтрація витрат;
   * перегляд залишку бюджету;
   * формування звіту за категоріями.

4. **Збереження даних**

   * Після кожної зміни дані автоматично зберігаються у файл.

5. **Контроль бюджету**

   * Після додавання витрати бот перевіряє, чи не перевищено бюджет.
   * Якщо витрати більші за бюджет, виводиться попередження.

6. **Завершення роботи**

   * Після введення команди `вийти` програма завершує роботу.

---

## Опис структури даних

Дані зберігаються у файлі **JSON** у вигляді словника.

Приклад структури файлу:

```json
{
    "budget": 1000,
    "expenses": [
        {
            "amount": 100,
            "category": "Їжа",
            "date": "2026-03-10",
            "comment": "Продукти"
        },
        {
            "amount": 50,
            "category": "Транспорт",
            "date": "2026-03-11",
            "comment": "Метро"
        }
    ]
}
```

Опис полів:

* **budget** — загальний бюджет користувача.
* **expenses** — список витрат.

Кожна витрата містить:

* **amount** — сума витрати;
* **category** — категорія витрати;
* **date** — дата витрати;
* **comment** — короткий коментар (необов’язковий).

---

## Інструкція зі запуску програми

1. Встановити **Python 3** на комп’ютері.
2. Завантажити або створити файл з програмою, наприклад:

```
expense_bot.py
```

3. Відкрити термінал або командний рядок у папці з програмою.

4. Запустити програму командою:

```
python expense_bot.py
```

5. Після запуску вводити команди для роботи з ботом.
